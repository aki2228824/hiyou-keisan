# 療養介護サービス提供実績記録票

入院患者が1か月に利用した日用品・食事などの費用を自動集計し、請求書（PDF）を出力する社内向けWebアプリケーションです。

---

## 主な機能

| 機能 | 内容 |
|------|------|
| **日次入力** | 患者ごとに日用品の使用数・食事回数・サービス提供状況を1か月分まとめて入力 |
| **月次集計** | 病棟・月を指定して全患者の費用を一覧表示 |
| **PDF出力** | 患者ごとの請求書をPDFで出力 |
| **マスタ管理** | 患者・物品・日用品単価・ユーザーの登録・編集・削除 |
| **ロール制御** | 管理者／一般スタッフ／閲覧のみ の3段階アクセス制御 |

---

## 画面構成

```
├── 日次入力
│   ├── 病棟選択（2階 / 3階 / 4階）
│   ├── 患者選択
│   └── 1か月分の日次入力テーブル
│       ├── サービス提供の状況・入院外泊時加算・入院時支援特別加算
│       ├── 朝食・昼食・夕食（回数）
│       └── 日用品項目（入浴〜洗濯 など）
│
├── 月次集計
│   ├── 病棟選択
│   └── 患者別 食事費・日用品費・合計金額一覧
│
└── マスタ管理
    ├── 患者管理
    ├── 物品管理
    ├── 日用品単価（月ごと設定・翌月以降引き継ぎ）
    └── ユーザー管理
```

---

## 技術スタック

| 区分 | 技術 |
|------|------|
| サーバー | Node.js + Express |
| データベース | sql.js（SQLite WebAssembly） |
| フロントエンド | Vanilla HTML / CSS / JavaScript（SPA） |
| 認証 | express-session + bcryptjs |
| PDF生成 | Puppeteer（サーバーサイドレンダリング） |

## セットアップ

### 必要環境

- Node.js 18 以上

### インストールと起動

```bash
git clone https://github.com/aki2228824/hiyou-keisan.git
cd hiyou-keisan
npm install
node server/index.js
```

ブラウザで `http://localhost:3000` にアクセスしてください。

### 初期ログイン情報

| ユーザーID | パスワード | ロール |
|-----------|-----------|--------|
| admin | admin123 | 管理者 |

> ログイン後、マスタ管理 → ユーザー管理からパスワードを変更してください。

---

## 社内サーバーでの運用

1. サーバー機に Node.js をインストール
2. リポジトリをクローンし `npm install` を実行
3. `node server/index.js` を起動（常駐化には [PM2](https://pm2.keymetrics.io/) を推奨）
4. 社内ネットワーク内のPCから `http://<サーバーのIPアドレス>:3000` でアクセス

## データについて

- データベースは `server/data/db.sqlite` に保存されます
- バックアップは同ファイルをコピーするだけで可能です
- 日用品単価は月ごとに設定でき、未設定の月は直近の設定値を自動引き継ぎします
- 食事単価・食事費上限は患者ごと・月ごとに設定可能です
