## 詳細テスト結果

### N-01　正しいID・パスワードでログインできる

**リクエスト：** `POST /api/auth/login {"username":"admin","password":"admin123"}`  
**期待値：** HTTP 200、ユーザー情報がJSONで返る  
**結果：** ✅ 合格  
**レスポンス：**
```json
{"id":4,"username":"admin","role":"admin"}
```

### N-02　ログアウトできる

**リクエスト：** `POST /api/auth/logout` → `GET /api/auth/me`

**結果：** ✅ 合格
